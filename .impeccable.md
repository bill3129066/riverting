# Riverting Design Context — Light Editorial System

## Product & Audience

- **Product**: AI Agent Marketplace with per-second streaming payments on X Layer.
- **Core actors**:
  1. Curators publish agents and monetize usage.
  2. Users discover and run agents, paying per-second.
  3. Platform hosts execution and settles proofs/payments.
- **Primary users**: DeFi traders, AI enthusiasts, hackathon judges.
- **User context**: fast decision-making, trust-sensitive financial interactions, high information density.

## Brand Personality

- **Precise**: no ornamental ambiguity; information architecture is explicit.
- **Editorial**: typography-led hierarchy and composition over UI chrome.
- **Trustworthy**: restrained color usage, high contrast, and stable visual rhythm.

## Aesthetic Direction

**Light Editorial** — clean, airy, magazine-like. High-contrast black typography on light surfaces. Sharp geometry (0px border-radius). Indigo accent for interactive/secondary emphasis. Newsreader serif for headlines (italic for drama), Inter sans for body/UI.

Think: Monocle magazine × Bloomberg editorial × Stripe documentation clarity.

## Non-Negotiable Design Principles

1. **Typography drives hierarchy, not boxes**.
2. **Whitespace is structural, not decorative**.
3. **Color is functional, not decorative**.
4. **Motion signals state, not decoration**.
5. **Sharp corners everywhere** — no rounded rectangles.

## Color System (Exact Tokens)

Light mode — clean, airy surfaces with black primary text.

- `background`: `#f9f9f9`
- `surface`: `#f9f9f9`
- `surface-elevated`: `#ffffff`
- `surface-dim`: `#f3f3f3` (container-low equivalent)
- `border-subtle`: `#e2e2e2`
- `border-strong`: `#c6c6c6`
- `text-primary`: `#1b1b1b`
- `text-secondary`: `#474747`
- `text-tertiary`: `#777777`
- `accent`: `#4a3bf6` (indigo/purple)
- `accent-muted`: `#d3d0ff`
- `accent-foreground`: `#ffffff`
- `success`: `#2e7d32`
- `warning`: `#e65100`
- `error`: `#ba1a1a`

### Usage Constraints

- Primary color is **black** (#000000) for key actions and the wordmark.
- Accent (indigo) is for secondary emphasis, active states, live data highlights, and interactive elements.
- Body copy defaults to `text-primary` (#1b1b1b); supporting copy uses `text-secondary` then `text-tertiary`.
- Borders default to `border-subtle`; use `border-strong` for higher-separation grid lines.
- Status colors (`success`, `warning`, `error`) are semantic-only, never decorative.
- **No rounded corners** — borderRadius: 0px on all elements.

## Typography System (Exact Fonts)

- **Display/Headings**: `Newsreader` (Google Font, italic for emphasis)
- **Body/UI/Labels**: `Inter` (Google Font)

### Type Intent

- Headings: editorial serif, dramatic at large sizes, italic variants for personality.
- Body/UI: clean sans-serif, precise, legible.
- Labels: uppercase, tracking-widest, small size — editorial section markers.
- Avoid generic AI-dashboard typography. Use italic serif headlines as the primary differentiator.

## Spacing Scale

Base scale (4px unit). Use generous px values for editorial breathing room:

- Sections separated by 96-192px (`mb-24` to `mb-48`)
- Content padding: 96px horizontal on desktop (`px-24`)
- Max width: 1920px
- Grid: 12-column with large gaps (`gap-8` to `gap-24`)

Rhythm rule:

- Tight spacing **within** a data group.
- Dramatic spacing **between** sections for editorial weight.
- Use whitespace as the primary information boundary before adding borders.

## Component Patterns

1. **Editorial Sections**: Section kicker (tiny uppercase tracking-widest) + serif headline + thin rule + content
2. **Bordered Grid**: Technical specs in bordered cells (no cards, just grid lines)
3. **Definition Rows**: Label/value with strong typographic contrast
4. **Split-Column Narratives**: 12-col grid, asymmetric (8+4, 4+8, 4+4+4)
5. **CTA as bordered text links**: Not pill buttons — underlined or border-bottom links with arrow icons
6. **Material Symbols** for icons (outlined, weight 300)

Anti-patterns to avoid:

- Rounded rectangles (borderRadius > 0)
- Card-inside-card nesting
- Teal/cyan accent colors
- Dark mode with glowing accents
- Emoji as section markers
- Identical card grids
